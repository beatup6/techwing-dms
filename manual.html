시스템 개요
📌 TECHWING DMS란? ISO 9001 기반 사내 표준 문서를 전자적으로 작성·검토·승인·배포·관리하는 통합 전자 문서 관리 시스템입니다. 모든 문서 이력이 자동 기록되며, 다단계 검토 및 AI 보조 기능을 통해 문서 품질을 보장합니다.

🌐

  • 사번 / 비밀번호 입력 후 로그인
  • 최초 로그인 시 비밀번호 변경 권장

단계별 상세 절차
1

문서 작성 — 각 부서 담당자

DMS에서 문서 등록 후 사내 표준 템플릿에 맞게 작성

수행 절차

  • DMS 로그인 → 문서 등록 메뉴
  • 문서 유형 선택 (절차서/지침서/서식/표준서)
  • 문서번호 자동 채번 또는 수동 입력
  • 제목, 소관부서, 정기검토일 입력
  • 파일 첨부 (PDF/DOCX)
  • 1차 검토 요청 버튼 클릭

작성 기준

  • 사내 표준 템플릿 양식 사용 필수
  • 목적, 범위, 책임과 권한 명시
  • 용어 정의 포함 (해당 시)
  • 업무 흐름도 또는 절차 단계 기술
  • 관련 문서 및 양식 번호 기재

주의사항

  • 임시저장으로 초안 보관 가능
  • 검토 요청 전 AI 사전 검토 권장
  • 파일명 규칙: [문서번호]_[제목]_vX
  • 개인정보 포함 내용 삽입 금지
2

1차 검토 — 작성자 부서 팀장

자동 알림 수신 후 내용 검토, 보완 요청 또는 2차 검토 승인

수행 절차

  • 알림 수신 → 결재함 메뉴 접속
  • 해당 문서 클릭 → 내용 검토
  • 문서 열람 및 첨부파일 확인
  • 승인 또는 보완 요청 선택
  • 보완 요청 시 사유 입력 필수

검토 항목

  • 내용의 정확성 및 부서 업무 반영 여부
  • 용어 통일성
  • 양식 준수 여부
  • 실제 업무 흐름과의 일치 여부

시스템 자동 처리

  • 작성자 1차 검토 요청 시 팀장에게 즉시 알림
  • 승인 시 자동으로 그룹장에게 2차 검토 요청 알림 발송
  • 보완 요청 시 작성자에게 사유 알림 발송
3

2차 검토 — 작성자 부서 그룹장

팀장 검토 완료 후 자동 알림, 부서 관점의 상위 검토 수행

수행 절차

  • 알림 수신 → 결재함 메뉴 접속
  • 팀장 검토 의견 함께 확인
  • 문서 내용 및 방향성 검토
  • 승인 또는 보완 요청 선택

검토 항목

  • 부서 전략 방향과의 일치 여부
  • 타 부서와의 업무 연계 적합성
  • ISO 요건 충족 여부 (사전 확인)
  • 책임과 권한의 적절성

시스템 자동 처리

  • 승인 시 사내표준관리 담당자에게 자동 알림
  • 보완 요청 시 작성자 및 팀장에게 알림 발송
  • 검토 이력 자동 기록
4

사내표준관리 검토 — 사내표준관리 담당자 + AI 검토

ISO 규정 적합성, 오타, 업무 흐름 일치 여부 등 종합 검토 (AI 보조)

수행 절차

  • 알림 수신 → 결재함 메뉴 접속
  • AI 검토 실행 버튼 클릭
  • AI 검토 결과 확인 (4개 항목)
  • AI 제안 내용 반영 여부 결정
  • 최종 검토 의견 입력 후 승인

AI 자동 검출 항목

  • ① 오타 및 잘못 기재된 내용
  • ② 본문과 업무 흐름 불일치
  • ③ ISO 9001 규정 미충족 항목
  • ④ 책임/권한 부서 검토 사항
  • ⑤ 각 항목별 모범 답안 제안

시스템 자동 처리

  • 승인 시 유관 부서 그룹장 전원 알림
  • AI 검토 결과 문서 이력에 자동 저장
  • 보완 요청 시 작성 부서 전 단계 알림
5

유관 그룹장 검토 — 관련 부서 그룹장 (실시간 댓글)

실시간 댓글 방식으로 의견 제시, 검토 확인 서명, 전원 완료 시 다음 단계 진행

수행 절차

  • 알림 수신 → 결재함 메뉴 접속
  • 문서 내용 열람
  • 의견 있으면 댓글 입력
  • 검토 완료 시 검토 확인 버튼 클릭
  • 전원 확인 완료 시 최종 승인 단계로 자동 이동

댓글 기능

  • 실시간으로 다른 그룹장 댓글 확인 가능
  • 댓글 작성자·시간 자동 기록
  • 댓글에 답글(Thread) 가능
  • 완료 여부 아이콘으로 시각 확인

시스템 자동 처리

  • 미검토 그룹장 현황 실시간 표시
  • 전원 검토 완료 시 임원 자동 알림
  • 댓글 및 검토 이력 전체 보존
6

최종 승인 및 배포 — 총괄 임원

그룹웨어 연동 결재 상신, 최종 승인 후 전사 자동 배포

수행 절차

  • 그룹웨어 결재함에서 승인 또는 DMS에서 직접 처리
  • 전체 검토 이력 및 댓글 확인
  • AI 검토 결과 최종 확인
  • 최종 승인 버튼 클릭
  • 배포 대상 부서 선택 후 배포

배포 후 자동 처리

  • 배포 대상 전 직원 알림 발송
  • 열람 확인 현황 실시간 추적
  • 구버전 자동 폐지 처리
  • 문서 이력에 승인자 정보 기록

그룹웨어 연동

  • 결재 URL 그룹웨어로 자동 전송
  • 그룹웨어에서 승인 시 DMS 자동 상태 변경
  • 미연동 시 DMS 내 직접 결재 가능

AI 검토 기능 사용법
⚡ AI 검토란? 사내표준관리 검토 단계에서 Anthropic Claude AI가 문서를 자동 분석하여 오류·불일치·ISO 미충족 항목을 검출하고 수정 방안을 제안합니다.

① 오타·잘못 기재 검출

  • 맞춤법 오류 자동 감지
  • 단위·수치 오기재 검출
  • 용어 불통일 지적
  • 수정 제안 텍스트 함께 표시

② 업무 흐름 불일치 검출

  • 절차 순서의 논리적 오류
  • 본문 설명과 흐름도 불일치
  • 누락된 단계 또는 중복 단계
  • 조건 분기 처리 누락

③ ISO 규정 적합성 검출

  • ISO 9001:2015 요건 체크
  • 필수 항목(목적/범위/책임) 누락 감지
  • 문서 번호 체계 규칙 확인
  • 개정 이력 기재 방식 확인

④ 모범 답안 제안

  • 각 지적 항목별 수정 예시 제공
  • ISO 준거 표현 방식 제안
  • 유사 기존 문서 참조 제안
  • 제안 채택·무시 선택 가능

⑤ 책임·권한 검토 정리

  • 책임 부서 명확성 분석
  • 권한 범위 적절성 검토
  • 부서 간 업무 중복·공백 지적
  • RACI 매트릭스 자동 생성

FAQ / 자주 묻는 질문
Q

검토 단계에서 보완 요청을 받으면 어떻게 하나요?

알림을 통해 보완 요청 사유를 확인하고, 문서 목록에서 해당 문서를 열어 수정 후 다시 검토 요청하면 됩니다. 이전 단계부터 다시 시작됩니다.
Q

AI 검토 결과가 마음에 들지 않으면 무시해도 되나요?

AI 제안은 참고용입니다. 담당자 판단에 따라 채택 또는 무시를 선택할 수 있으며, 무시 사유를 간략히 입력하는 것을 권장합니다. 단, ISO 필수 항목 미충족은 반드시 수정 후 진행해야 합니다.
Q

그룹장 검토에서 한 명이 오랫동안 완료하지 않으면?

결재함에서 미완료 그룹장 목록을 확인하여 "독촉 알림" 버튼으로 알림을 재발송할 수 있습니다. 관리자는 부득이한 경우 대리 처리도 가능합니다.
